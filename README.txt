This Java program demonstrates various search algorithms applied to both ArrayList and LinkedList data structures using a dataset of articles. Additionally, the program conducts empirical testing to analyze the efficiency and performance of these search algorithms.

-------------------------------------------------------------------------------------------------------------------------------------
Overview:

The program reads article data from a CSV file and stores it in both ArrayList and LinkedList data structures. It offers a user-friendly interface for conducting searches using different algorithms and comparing their performance. The program also facilitates empirical testing to evaluate the efficiency of the search algorithms across various scenarios.

-------------------------------------------------------------------------------------------------------------------------------------
How to Run:

Compile the Java source code using a Java compiler (e.g., javac Source.java).
Run the compiled program (e.g., java Source).


-------------------------------------------------------------------------------------------------------------------------------------
Program Functionality
The program provides the following features

Search Algorithms:

Linear Search
Binary Search (requires sorting)
Sentinel Linear Search
Jump Search


Options:

Search in ArrayList or LinkedList
Print the contents of ArrayList or LinkedList
Compare algorithms' performance
Perform empirical testing

-------------------------------------------------------------------------------------------------------------------------------------
Usage Instructions:

Upon running the program, follow the on-screen prompts to select options.
Perform searches, compare algorithm performance, and analyze empirical results.
The program provides clear instructions and outputs to guide you through the process.

File Description:

Source.java: Main Java program source code.

Additional Notes:

The program ensures input validation and robust functionality.
